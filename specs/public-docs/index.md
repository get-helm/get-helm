# HELM — Your Personal Automation Platform

HELM turns your Discord server into a command center for an AI team that works for you around the clock.

---

## What is HELM?

HELM is a self-hosted automation platform that runs on your Mac Mini (or always-on Mac) and connects to your Discord server. Instead of one AI chatbot, you get a coordinated team of specialized agents — each one focused on a specific job — that handle real tasks, report back in plain English, and escalate decisions only when they need you.

### What it does today

- **Morning briefings** — Daily summary of what matters, delivered to your Discord at the time you choose
- **Workspaces** — Focused agent environments for recurring tasks: ETF analysis, options trading review, financial summaries, and more
- **Proactive monitoring** — Agents that watch for patterns, anomalies, and opportunities in your data
- **On-demand automation** — Type a request in Discord, get real work done within minutes
- **Second brain** — Captures your notes, decisions, and ideas; surfaces them when relevant

### Who it's for

HELM is for people who want AI working *for* them, not just answering questions. If you're comfortable running a small server and want automation that fits your actual life — not a generic assistant — HELM is for you.

You don't need to write code to use HELM. You do need:
- A Mac Mini or always-on Mac (the bot host)
- A Discord account and server
- About 30 minutes to set up

---

## How it works

```
You → Discord → HELM Bot → Specialized Agent → Work gets done → Report back to you
```

Every message you send to your Discord is routed to the right agent. Agents work autonomously, report progress, and stop for your input only when a decision requires it.
