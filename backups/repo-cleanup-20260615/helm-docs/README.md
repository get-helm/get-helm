## What does HELM do?

HELM sits in your Discord server and automates things for you:

- **Daily digests** — summarize your emails, portfolio, news
- **Alerts** — notify you when something important happens
- **Data tracking** — pull from APIs and organize data automatically
- **Scheduled tasks** — run things on a schedule without thinking about it
- **Research** — gather and synthesize information for decisions

You describe what you want in plain English. HELM handles the rest.

---

## Quick Example

In Discord, type:

```
@HELM send me a daily summary of my ETF portfolio
```

HELM will:
1. Confirm what you want
2. Ask clarifying questions
3. Design the solution
4. Build and test it
5. Run it automatically on your schedule
